# 🍃 ReciclaTech

## 📌 Sobre o Projeto
Este repositório contém o código-fonte do Sistema ReciclaTech, desenvolvido como requisito avaliativo de extensão para as disciplinas de Engenharia de Software, WEB I, ministrado professores Kelvin de Oliveira e Pedro Lucas, no Instituto Federal do Ceará (IFCE).

O objetivo principal deste projeto é criar uma plataforma que ajude a concientizar, e administrar doaçoes de aparelhos não utilizados. O sistema permite o cadastro de usuários, o registro de doações e o acompanhamento das arrecadações, aplicando na prática os conceitos de Engenharia de Software, e WEB I.

## ⚙️ Funcionalidades
* **Cadastro de Usuários:** Criação de perfis para Usuárioss.
* **Gestão de Contatos:** Armazenamento de e-mails e telefones dos usuários.
* **Registro de Doações:** Vinculação de uma doação específica de um doador.
* **Controle de Arrecadação:** Atualização automática do saldo arrecadado pelas instituições, e a efetividade de um ranking de doações, e reciclagens.

## 💻 Tecnologias Utilizadas
* **Linguagens:** JavaScript, PHP, HTML, CSS
* **Banco de Dados: MySQL**
* **Ferramentas Auxiliares:** Bootstrap, XAMPP
* **Paradigma:** Funcional
* **IDE Recomendada:** VS Code
* **Modelagem:** Diagrama UML - draw.io, brModelo
